Detecting Whales Using Drone Imagery

Project Details

The client had 200 pictures of beluga whales captured from a drone. The goal was to locate the whales in the images. I trained a YOLOv5 object recognition network to draw bounding boxes around the beluga whales with impressive results.

This model was later used for transfer learning to track different species of whales taken from drone footage. The gif on the right shows the results from the final model.

The difficulty of this project was the small dataset. I employed several techniques such as transfer learning and data augmentation in order to get as much performance out of the model as possible.

ChatGPT Discord Bot Integration

Gallery Description

I was tasked with integrating chatGPT to a Discord bot in development. I used the OpenAI API with NodeJS and Python to read user prompts and send the reply to Discord's chat. I included custom system setup for chatGPT to understand the name of the bot and it's function to provide entertaining responses. Because of Discord's character limit, responses exceeding the character limit are split into several messages.

The bot has short-term memory and is able to recall previous recent conversations. It is also able to distinguish between different users and understand separate chat history between users (for example assigning a nickname to users or responding in certain languages to certain users)

Holographic Camera Project

Gallery Description

Oil underwater is often in the form of micro-sized oil droplets, making it very difficult to quantify underwater oil spills. Autonomous Underwater Vehicles are being used to monitor oil spills in the ocean. These vehicles use a special camera called the holographic camera to capture images of these oil droplets. Special software can process these images to provide different focal planes from a single image. While useful to capture several several oil droplets at once, it is difficult to measure and quantify the oil droplets because of the processing time and difficulty in processing these images.

My thesis focused on methods to speed up processing these images. Three methods to process these images were developed:

  • Traditional computer vision technique: This technique used edge detection to find the oil droplets
  • Semantic Segmentation ML Model: This technique used a U-Net to segment the images based on supervised learning
  • YOLOv5 Object Detection ML Model: This technique used a YOLO
model to detect the oil droplets and draw a bounding box around the droplet.

Compared to the traditional computer vision technique, the machine learning techniques found more particles and had a speed up of 250% in execution time.
